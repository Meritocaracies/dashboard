Connect Google Calendar

Enter your Google Calendar API key and Calendar ID to see your events directly in the dashboard. Your credentials stay in your browser only.

1. Go to console.cloud.google.com
2. Enable Calendar API → Credentials → Create API Key
3. Restrict key to "Google Calendar API"
4. Your Calendar ID is in Calendar Settings (usually your Gmail address)
